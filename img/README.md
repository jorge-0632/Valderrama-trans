## 💾 CÓMO AGREGAR LAS IMÁGENES:

1. Guarda tus imágenes JPG o PNG con los nombres exactos listados arriba
2. Coloca todas las imágenes en esta carpeta (`img/`)
3. Las imágenes se cargarán automáticamente en la web

**IMPORTANTE:** Los nombres deben ser exactamente como se muestran (mismo caso, extensión, etc.)
